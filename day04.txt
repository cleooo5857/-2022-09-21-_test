조건문(if문, switch문)
반복문
    : 특정한 코드나 행위를 반복해야할 때 사용

    while문
        -조건에 의한 반복

        while(조건식){
            ... 조건식이 true일 동안 반복할 문장
        }

        do ~ while문
            조건식의 결과와 상관없이 한번은 반드시 실행하는 while문

            do{
                ...조건식이 true일 동안 반복할 문장
            }while(조건식)
        
        무한루프(무한반복) // setinterval es6에 출현 후 잘 사용하지않음
            while(true or 항상 조건이 true인 경우){
                ...
                if(탈출 조건식){
                    ...
                    break
                }
            }

    for문 
       반복 횟수가 특정된 경우에 사용

       ex)
        for(초기식; 조건식; 증감식){
            ...조건식이 true일 동안 반복할 문장
        }

            for(let i=0; i<5; i++){
                ...
            
            }
            초기식: 변수로 표현 (내부에서 변하는 값을 표현 가능)
            조건식: 반복 수행 될 조건식
            증감식: 매 반복마다 마지막으로 수행할 문장
